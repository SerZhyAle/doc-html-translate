Создать программу вероятнее всего на языке Гол. Рама не имеет интерфейса. При ее запуске ex ф файла просто так. Она регистрируется в Windows для того, чтобы быть программой по умолчанию для файлов ePUB. Достаточно сообщение о том, что программа зарегистрирована. При нажатии на Епап открытие Виндоус должна срабатывать программа для этого файла. Логика следующая программа открывает epab файл и создает в В том же каталоге подкаталог по имени этого файла.В котором создает Html структуру, полностью отвечающую внутренностям Епаб ф файла с картинками и шрифтами и так далее. То есть в этом подкаталоке уже будет индекс Html, который при открытии в браузере будет показывать Содержимое..
Но это не все. После того, как такой подкаталок содержимое будут созданы, будут использоваться мои creds для Google Для того, чтобы перевести содержимое на русский язык. После перевода содержимого и обновления Html страниц Программа запускает хром для этого файла, то есть открывает его для чтения.
У программа будут флаги. Да, по умолчанию она делает предыдущее поведение. Но определенным флагом можно, например, отключить перевод. Другим флагом заказать другой язык источника определенный, не автоопределенный. И третьим флагом заказать другой язык перевода, не русский.
