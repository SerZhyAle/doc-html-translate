Open EPUB/PDF in Chrome and translate to your language for free.

Convert books and documents to local HTML, keep navigation, then use Chrome page translation. This is the fastest no-cost reading workflow.

# Fast free workflow doc-html-translate.exe -notranslate "book.epub" # or # doc-html-translate.exe -notranslate "book.pdf" # tool opens index.html in browser # in Chrome: Translate page -> your language
Step 1

Convert Without API

Run with -notranslate for instant extraction and clean HTML pages.

Step 2

Open In Chrome

Read from generated index with TOC and page navigation.

Step 3

Translate In Browser

Use Chrome built-in page translation to switch to any target language.

Supported Input Formats

EPUB, PDF, TXT, Markdown, FB2, RTF, HTML are supported.

EPUB PDF TXT MD FB2 RTF HTML
